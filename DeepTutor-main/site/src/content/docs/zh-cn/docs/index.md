## DeepTutor 是什么

一个开源、agent-native 的学习伴侣。**五种模式**（Chat / Solve / Quiz / Research / Visualize）共享同一个 session、知识库和引用历史。**TutorBot** 把你的上下文带到 Telegram、Slack、飞书等渠道。**三层记忆系统**（含可审查的 Memory Workbench 和 Memory Graph）让你随时审计 *为什么* 系统会记住某件事。

由香港大学数据智能实验室构建。Apache 2.0 协议。
